## Exponentially weighted moving average
This is also called simple exponentail smoothening.   
$y_t = \sum_{i=0}^{t}w_ix_{t-i}/\sum_{i=0}^{t}w_i$  
How you will define the weight term depends on the value of the adjust parameter. when set to True, the weights are calculated by dividing with the decaying adjustment factor in beginning periods to account for imbalance in relative weightings. This parameter is True by default and if left unchanged following will be the formula 
$y_t = x_t + (1-\alpha)x_{t-1} + (1-\alpha)^2x_{t-2} + (1-\alpha)^3x_{t-3} + (1-\alpha)^tx_{0} / 1 + (1-\alpha) + (1-\alpha)^2 + (1-\alpha)^3 + (1-\alpha)^t$  
$\alpha = 2/(span + 1)$ 
you can use the ewm() function in Pandas to calculate exponentially weighted moving average, alpha or make you life easier with the span parameter. 
## Stationarity Test and Automation   
A stationary series is centered around some value, doesnt have too many spikes and unexpected variations and doesnt show drastic behaviour changes from one part to another.  
1. Stationary series are easy to analyze 
2. Most forecasting algorithms assume a series is stationary.
