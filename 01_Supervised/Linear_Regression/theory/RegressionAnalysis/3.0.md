## Polynomical Regression 
Here also we will apply the same Linear regression principle only thing extra is we will add the polynomial terms, we will increase the degree of features and add it as a feature   
$y = \beta_0 + \beta_1*x_1 + \beta_2*x_1^2 + \beta_3*x_2 + \beta_4*x_2^2$  
x1 and x2 are the features    
with higher value of degree model will try to overfit and with lower value of degree model will underfit.
