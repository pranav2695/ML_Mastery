## F-statistics and Prob (F-statistics)
It is used to determine whether a linear regression mode is statistically significant , meaning it provide better fit to the data than just using the mean of the dependent variable
## Steps for the F-test for overall significance (this is the variation of annova)
1. Define the null and alternative hypothesis 
for the linear relation we will have   
$y = \beta_0 + \beta_1*X$  
H0 : $\beta_1$ = 0 Ha : $\beta_1$ != 0  
2. calculate the F-statistics = MSR/MSE  
MSR = ESS/K, MSE = RSS/(n-k-1)  
MSR = (TSS - RSS)/k
3. For the F-statistics we will calculate the P-value if p- value < 0.05 then we will reject the null hypothises. 
4. F-statistics is ratio of explained and unexplained error if this ration is high explained error is greater than unexplained error hence p value for that value will be small hence we will reject the null hypothesis.  
if we have multiple coefficient then we can say atleast one of them is not zero, but wont be able to tell which is zero or which are not zeros. 
### Points for test 
- if we square the normal distribution number we will get chi sqaure distribution and if we divide the chi sq with degree of freedom and take ratio of two different distribution then it belongs to F distribution 
## To find how strong the relationship check R2 square
$R2 = ESS/RSS$  
$R2 = 1 - (RSS/TSS)$
