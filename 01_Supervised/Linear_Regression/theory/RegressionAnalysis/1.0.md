        - irreducible error: for eg we are predicting the LPA of student using CGPA and iq but still we wont be able to predictly exactly how much we will get , there are so many other factors on which it will depend, so these error which we can minimize are called irreducible errors
        $LPA = f(CGPA, IQ) + error$

    - Reducible error: $f() - f'()$
