2. Insufficient Data/Labelled Data
- unreasonable effectiveness of the data, if we dont have the labelled data
3. Non representative data

4. Poor quality data
5. Irrevelant  features
- garbage in garbage out, we only require the good features which have good co relation with the predictor 
- we can combine the features called feature engineering
6. overfitting 
- Memorize the train data by the model
7. Underfitting
- Not able to learn the train data proper
8. software Integration 
9. Offline Learning / Deployment 
10. Cost Involved
