# Multiple Linear Regression

As we will have multiple features so now we will not find the line equation we will find the hyperplane equation in n-dimensional co ordinate system  
$y = \beta_0 + x_1\beta_2 + x_2\beta_2 .... x_n\beta_n$
## Mathmetical Formulation 
By using the Matrix we can drived the one equation which will hold true irrespective of number of dimension  
$y_{hat} =  X\beta$
shape is n * 1, n --> no. of row and 1 columns  
Now the loss function in the matrix form is 
$e^Te$  
In the linear regression we will have one global minima and no local minima.
