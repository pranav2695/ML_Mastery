## Linear Regression
It is the supervised ml algorithm
3 types of LR
1. simple LR (1 input col)
2. Multi LR (More than 1 input col)
3. Polynomial LR (when the data is not Linear then we use polynomial regression)
## Simple LR
- Data is sort of linear not 100% linear, because of n number of factor, and this error which we cant quantify called as stohastic errors.
- so in the LR we will create the line which do the minimum errors for all the data points thats why it is called best fit line.
- In the LR we have coef_ called weightage and offset, weightabe tell us how much dependent variable is dependent on the input col and offset is, even when input variable will be zero, we will predict some values which is the offset.
- In the scikit-learn LR class OLS is implemented whereas if you want to use the Gradient descant then we can use the SGDRegressor.
