## MLDLC
1. Frame the problem: Correctly frame the problem, cost of the project, team required, end to end complete requirement
2. Gathering the data: csv files, fetching data from api, web scraping, data warehouse (ETL) using spark for the bigdata
2. Data processing: remove duplicates, remove missing values, remove outliers, scale the values. core idea is to convert the data into format in which it can be easily consumed 
4. EDA: Vizs, bivariate, univaraite, outlier detection , imbalanced data.
5. feature eng and selection : create new features, select the important features
6. Model training/evaluation and selection: bring different models and evaluate which model to select
after selecting the model we do the hyperparameters tuning of the best model that we selected.
7. Model deployment: we create the binary file using the tool like pickle and then deploy the model on the server and when we give the correct input to this file it will return the output in the json format to the user
8. Testing: A/B testing 
9. Optimize: backup the model and data, automation for rollback, load balancing , how freq need to train (called rotting), optimize where we have more cost.
