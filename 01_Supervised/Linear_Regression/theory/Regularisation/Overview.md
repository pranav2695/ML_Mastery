# Regularization 
It is the simple technique to reduce the overfitting, it is not specific to one model, It is the universal approach.  
## When to use Regularization 
1. Preventing Overfitting: Regularization is most commonly used as tool to prevent overfitting. if your model performs well on the training data but poorly on the validation or test data, it might be overfitting, and regularization could help.  
2. High Dimensionality: It is also useful when we have more number of features compared to data, in those cases model will overfit very fast. 
3. Multicollinearity: It can help by distributing the coefficient estimates among correlated features.  
4. Feature selection: we can use the Lasso L1 to make the feature coeff to zero.  
5. Interpretability: It can help by constraining the models complexity 
6. Model performance: It might still improve your model out of sample prediction performance. we can experiment. we can always apply the regularisation.
