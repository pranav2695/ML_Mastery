2. Hardware issue as we need to do the computation with all the traning data


## Advantage of stohastic GB
1. Faster convergence 
2. we will select the row randomly


## Disadvantage 
1. it will not give the steady result

## when to use the stohastic GD
- for the big data 
- if you have the non convex function: we can stuck in the local minima in case of batch GD, But in case of Stohastic GD we will not stuck in the local minima.
