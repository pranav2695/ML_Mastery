## Sigmoid function 
$y = 1/(1+e^{-z})$  
Instead of using the step function which will give the 0 or 1 if the data point lies above or below the line. we will use the sigmoid function if it is in positive region of line it will be greater than 0.5 and value will be higher for higher positive number and vice versa for negative points.
