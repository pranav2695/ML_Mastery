## Sigmoid function

## Maximum likelihood  
The likelihood function is the product of the predicted probabilities for the actuals class of each observation.  
we want to maximize this number. as this value can become very close to zero so we take the log of all the product of proabilities and use that. we will take negative of the log so now we need to minmize this loss.  
Binary cross entropy/log loss error  
$Loss = 1/n\sum_{i=1}^{n}-y1\log(y_{hat})-(1-y1)\log(y_{hat})$
